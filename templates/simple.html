zip

Name

bpy zip - export a bpy folder to a zip archive

Synopsis

The zip command creates a zip archive out of the specified folder.

Usage

bpy zip [-at=TIMESPEC] src > src.zip

Example

Fetch the entire root as a zip archive:

$ bpy zip / > out.zip
