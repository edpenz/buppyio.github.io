mv

Name

bpy put - move a file or folder

Synopsis

The mv command lets you move a file or folder from one path to another.

Usage

bpy mv src dest

Example

Rename a file:

$ bpy ls
hello.txt
$ bpy mv hello.txt renamed.txt
$ bpy ls
renamed.txt
